# letslearnhangul.com
Circa 2016 source code for letslearnhangul.com

### Developing locally
- Use Node 6.16.0.
- Run `npm install` to install dependencies and build assets.
- Run `./node_modules/.bin/gulp watch` to recompile assets after changes.
- Run `npm start` to run the dev server -- serves compiled assets from `public`.
